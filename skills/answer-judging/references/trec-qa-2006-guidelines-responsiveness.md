# TREC 2006 QA Track Guidelines: "Responsive" Answers

Note: The TREC site hosts multiple QA-2006 guideline pages. This reference uses the main QA-2006 guidelines page used for judging definitions and examples.

URL: https://trec.nist.gov/data/qa/2006_qadata/qa.06.guidelines.html

Accessed: 2026-02-20

## Why This Source Matters

These guidelines explain how answers are judged in an established QA evaluation setting, including the notion that an answer can be deemed incorrect if it is *not responsive* to the question requirements.

## Notes (Paraphrased)

- The guidelines specify that an answer-string can be marked incorrect if it does not contain the correct answer or if it is not responsive to the question.
- "Responsive" includes meeting question constraints (e.g., providing units when asked, answering about the correct entity rather than a related one).

## Minimal Quotes (Verbatim, <= 25 words each)

- "incorrect: the answer-string does not contain a correct answer or the answer is not responsive;"
- "Being \"responsive\" means such things as including units for quantitative responses"
