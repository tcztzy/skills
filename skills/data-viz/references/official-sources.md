## Stable takeaways encoded in this skill

- Choose chart family before choosing plotting library.
- Reject common anti-patterns before generating code.
- Use static libraries for publication control, and HoloViz for browser-native interaction.
- Escalate to Datashader when ordinary plotting begins to hide the data distribution.
