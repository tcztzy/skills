# Little (1961): A Proof for the Queuing Formula: L = (lambda) W

## Source

- John D. C. Little (1961). *A Proof for the Queuing Formula: L = (lambda) W*. **Operations Research**, 9(3), 383-387. DOI: 10.1287/opre.9.3.383
- Landing page (abstract + bibliographic details): https://ideas.repec.org/a/inm/oropre/v9y1961i3p383-387.html

## What to use from it (for this skill)

- **Little's law:** In steady state, the long-run average number of jobs in the system (L) equals the effective arrival/throughput rate (lambda) times the average time a job spends in the system end-to-end (W): `L = lambda * W`.
- **Interpretation for shared queues:** If W increases (jobs taking longer end-to-end) at a given lambda, then L must increase too (more backlog/in-flight work), matching "submission surge/backlog increases latency" explanations.
