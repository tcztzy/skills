# Zotero Connector notes

Use these notes when capturing papers directly from publisher websites (for example Nature) into the local Zotero desktop app.

## Typical flow

1) Install **Zotero Connector** for your browser.
2) Keep **Zotero desktop** running.
3) On an article page, click the Connector button to save the item (and attachments when available).
4) (Optional) Select the destination **collection** in the Connector UI before saving.

## Useful behaviors

- If the page contains multiple items, the Connector typically shows a **folder** icon so you can pick which items to save.
- If Zotero desktop is closed, the Connector can save to **zotero.org** (if logged in) and later sync to the desktop app.

## Settings to know (when saving to zotero.org)

In the Connector preferences, there are options for what to save alongside metadata when importing to zotero.org, such as:

- Automatically take **snapshots**
- Automatically save **PDFs** (when available)
