# Zotero data directory notes

Use these notes to locate the Zotero data directory and understand its contents.

## How to locate

In Zotero, open Settings or Preferences, go to the Advanced tab, and use the Files and Folders section to click "Show Data Directory".

## Default locations

- macOS: ~/Zotero
- Windows: %USERPROFILE%\\Zotero
- Linux: ~/Zotero

## Contents

- zotero.sqlite: main database file
- storage/: attachment files organized by item key

## Caution

Close Zotero before moving or copying the data directory to avoid data loss or corruption.
