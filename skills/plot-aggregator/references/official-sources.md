## Stable takeaways encoded in this skill

- Export vector files for graphs when possible, and keep raster previews as companion artifacts.
- Use explicit export size and dpi instead of relying on interactive window defaults.
- Prefer perceptually uniform continuous colour scales and accessible discrete palettes.
- Treat labels, legends, captions, and error-bar definitions as part of the figure, not optional polish.
